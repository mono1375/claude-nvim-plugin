PURPOSE:
--------
Integrate Claude Code CLI into Neovim/LazyVim, providing AI assistance
directly in the editor without requiring a separate API key.

FEATURES:
---------
✅ Ask Claude questions from Neovim
✅ Get code suggestions
✅ Review entire files
✅ Explain selected code
✅ Generate code from descriptions
✅ Uses existing Claude CLI auth (no API key needed)

INSTALLATION:
-------------
./install.sh

Or manually:
cp plugin/init.lua ~/.config/nvim/lua/claude-mcp/init.lua
cp plugin-config.lua ~/.config/nvim/lua/plugins/claude-mcp.lua

USAGE:
------
<Space>ca  - Ask Claude a question
<Space>cs  - Get code suggestions
<Space>cr  - Review current file
<Space>cg  - Generate code
<Space>ce  - Explain selection (visual mode)

COMMANDS:
---------
:ClaudeAsk [question]      - Ask Claude
:ClaudeSuggest             - Get suggestions
:ClaudeReview              - Review file
:ClaudeGenerate [desc]     - Generate code

HOW IT WORKS:
-------------
Neovim Plugin → claude --print "question" → Claude API → Response in vsplit

Uses your existing `claude` CLI authentication.

REQUIREMENTS:
-------------
- Neovim 0.9+
- LazyVim
- Claude Code CLI (authenticated)
